# .impeccable — Mihrab Design Language

## Identity: Luminous Heritage
Sacred modernism meeting South Asian warmth. Geometric precision from Islamic tradition, rendered in contemporary CSS. Every detail is intentional.

## Color Philosophy
Warm cream (`#FAF6EE`) grounds every surface — never stark white. The brand teal (`#41C2DC`) is a jewel color used for interactive elements and accents, never as a background wash. Gold (`#C9922A`) appears only for prestige moments: labels, ornaments, badges. Deep navy (`#1A2744`) replaces pure black everywhere for softness and warmth.

## Typography Philosophy
Cormorant Garamond carries the soul — it has the weight of scripture, the elegance of manuscript. It belongs to every `h1`–`h3` and display text. DM Sans handles the data — legible, neutral, modern. It belongs to body copy, labels, form fields, and UI text. Never mix them at the same visual level. Never use Inter, Roboto, Arial, or system fonts as primary typefaces.

## Spacing Discipline
Base unit: 8px. All spacing is multiples of 4px minimum, 8px preferred. Section padding: 96px vertical on desktop, 64px on mobile. No arbitrary pixel values.

## Animation Philosophy
Motion should feel like the unfurling of a scroll or the opening of wooden mashrabiya screens. Entrance: fade up with slight scale (`revealCard` keyframe). Hover: gentle lift with warm shadow bloom. All animations must be wrapped in `@media (prefers-reduced-motion: no-preference)`. Never bounce, spin, or rush.

## Geometric Language
Islamic geometric patterns rendered via CSS: `repeating-linear-gradient` crosshatch, `clip-path` octagon shapes, gold hairline borders. The `.divider-arabesque` (line-diamond-line) separates sections. The `.shape-octagon` frames icons. The `.geo-watermark` adds subtle diagonal grid texture to dark sections.

## Surface Hierarchy (light to dark)
1. `--surface-cream` (#FAF6EE) — page ground
2. `--surface-warm-white` (#FFFFFF) — card ground
3. `--surface-muted` (#F0EBE1) — alternate section background
4. `--surface-dark` (#1A2744) — dark sections (footer, hero, CTA)
5. `--surface-darker` (#0F1A30) — navbar, deepest dark
6. Glass panels — white/72% or cream/85% with `backdrop-filter: blur(16px)`

## Rules — Do
- Use Cormorant Garamond for every h1–h3 and display heading
- Use the 8px spacing grid exclusively
- Add `.divider-arabesque` between section heading and content when page layout warrants it
- Apply `.animate-card` + stagger classes to all card grids
- Use `.text-gold-label` (uppercase, DM Sans, tracked) for section category labels above headings
- Use the `.shape-octagon` clip-path for all icon containers

## Rules — Do Not
- Never use stark white (#fff) as the page background — always `--surface-cream`
- Never use pure black for any text — always `--text-headline` (#1A2744) or `--text-body` (#3D3529)
- Never use teal as a background fill (only buttons, links, accents)
- Never put white text directly on `--brand-teal` (contrast fails)
- Never use `--brand-gold` for body copy (contrast ~3.2:1, decorative labels only)
- Never exceed `border-radius: var(--radius-xl)` (24px) on cards — only pill buttons use 999px
- Never use inline `onMouseEnter/Leave` for hover effects — use CSS only
